## Agent Installation:

1. Downloaded the linux agent from Veeam (https://www.veeam.com/products/downloads.html?ad=top-sub-menu)
2. Placed the .rpm file on the Fedora Server 
![](attachments/Pasted%20image%2020250213150925.png)
3. ssh into Fedora Server 
4. Went to /root/Veeam
5. Installed Veeam with these commands:
```
sudo dnf install ./veeam-release*.rpm -y
sudo dnf install veeam -y
sudo veeam

```
